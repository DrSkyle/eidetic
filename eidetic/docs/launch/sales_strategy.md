# Eidetic Sales Strategy

## 1. Product Positioning
**Slogan:** "The Filesystem That Remembers."
**Value Prop:** Eidetic transforms your static file storage into a dynamic, intelligent partner. It organizes your chaos, remembers every version, and secures your secretsâ€”all without leaving your local machine.

## 2. Target Audience
### Primary: Power Users & Developers
- **Pain Point:** Messy downloads folders, lost code snippets, accidental overwrites.
- **Hook:** "Time Travel" (Snapshots) and "Context Bundler" for LLM prompting.
- **Channels:** Hacker News, Reddit (r/rust, r/selfhosted, r/productivity), Twitter/X Tech bubbles.

### Secondary: Privacy Advocates
- **Pain Point:** Cloud storage scanning files, lack of local encryption.
- **Hook:** "The Vault" - transparent local encryption that works with any app.
- **Channels:** PrivacyTools.io, r/privacy.

## 3. Pricing Model
**Freemium Model:**
- **Free Tier:** Core features (Time Travel, Auto-Organization).
- **Pro Tier ($XX/year):** 
    - Unlimited "Vault" storage.
    - Advanced AI features (PDF OCR, Semantic Search).
    - Priority Support.

## 4. Launch Phases
### Phase 1: Soft Launch (The "Alpha")
- **Goal:** Gather feedback and fix bugs.
- **Action:** Release on GitHub as open source (core).
- **Metric:** GitHub Stars, Issue reports.

### Phase 2: The "Show HN" / Reddit Splash
- **Goal:** Viral traffic.
- **Action:** Post the "Reddit Launch Post" (see separate doc).
- **Asset:** Needs a cool GIF showing a file being saved and instantly "organized" or "versioned".

### Phase 3: Product Hunt
- **Goal:** General market awareness.
- **Action:** Scheduled launch day with hunted support.
